# Seagull Detection and Counting using Machine Learning

This project focuses on automatic detection of birds in images using machine learning and computer vision techniques.

The goal of the project is to build and compare several machine learning models for detecting birds in images.

---

## Dataset

Dataset: **Where are the seagulls? (Kaggle)**

The dataset contains images with seagulls that must be detected using computer vision methods.

The dataset was used to train and evaluate different machine learning models.

---
